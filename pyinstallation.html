step-1:

Open the Google Chrome.

In the search box (or) url, type Python download

step-2:

Click on the download option based on your computer configuration.

Example: windows, 64 bit, 32 bit, etc,.

step-3:

step-4:

Download Visual Studio Code to write the code.

step-5:

Create a folder inD drive and open it in the Visual Studio Code to start the coding.

How To Create a Folder:

  1. open the D drive in My Computer.
  2. click on D drive.
  3. right click on mouse (or) hold ctrl+shift+N to create a folder.
  4. folder will be created.

How To Open a Folder in Visual Studio Code:

  1. open the Visual Studio Code.
  2. click on file at top left corner.
  3. click on open folder.
  4. select the created folder in the D drive to open in the Visual Studio Code.
