Python Lists

List:
  • Lists are used to store multiple items in a single variable.
  • Lists are one of 4 built-in data types in Python used to store collections of data.
  • Lists are created using square brackets.
List Items:
  • List items are ordered, changeable, and allow duplicate values.
  • List items are indexed, the first item has index [0], the second item has index [1] etc.
List Ordered:
  • Lists follows an particular (or) given order and that order will not change.
  • If you add new items to a list, the new items will be placed at the end of the list.
Changeable (or) Mutable List:
  • The list is changeable, that means we can change, add, and remove items in a list after it has been created.
Allow Duplicates in List:
  • Lists can have items with the same value, same names, etc.

/* Accessing the list in python */

Code:

vegetables = ["potato", "carrot", "tomato", "brinjal"]

print(vegetables)

Output:

['potato', 'carrot', 'tomato', 'brinjal']

/* Accessing list with index */

Code:

print(vegetables[0])

print(vegetables[1])

Output:

potato
carrot

/* Adding items in the list */

Code:

vegetables.append("cauliflower")

vegetables.append("ladyfinger")

print(vegetables)

Output:

['potato', 'carrot', 'tomato', 'brinjal', 'cauliflower', 'ladyfinger']

/* Inserting items in the list at a particular index position */

Code:

vegetables.insert(4, "cabbage")

vegetables.insert(5, "bottleguard")

print(vegetables)

Output:

['potato', 'carrot', 'tomato', 'brinjal', 'cabbage', 'bottleguard', 'cauliflower', 'ladyfinger']

/* updating items in the list at a particular index position */

Code:

vegetables[1:3] = ["bitterguard", "radish"]

print(vegetables)

Output:

['potato', 'bitterguard', 'radish', 'brinjal', 'cabbage', 'bottleguard', 'cauliflower', 'ladyfinger']

/* removing items in the list from the last */

Code:

vegetables.pop()

vegetables.pop()

print(vegetables)

Output:

['potato', 'bitterguard', 'radish', 'brinjal', 'cabbage', 'bottleguard']

/* removing selected items in the list */

Code:

vegetables.remove("radish")

print(vegetables)

Output:

['potato', 'bitterguard', 'brinjal', 'cabbage', 'bottleguard']

/* reversing the items in the list */

Code:

vegetables.reverse()

print(vegetables)

Output:

['bottleguard', 'cabbage', 'brinjal', 'bitterguard', 'potato']

/* deleting all the list items */

Code:

vegetables.clear()

print(vegetables)

Output:

[]

/* zip is a method used to combine multiple elements into a single element (list) */

Code:

tech = ["html", "java", "css", "python"]

course = ["frontend", "backend", "frontend", "backend"]

result = list(zip(tech, course))

print(result)

Output:

[('html', 'frontend'), ('java', 'backend'), ('css', 'frontend'), ('python', 'backend')]

/* Number of duplicate items counts in the list */

Code:

vegetables.append("cabbage")

vegetables.append("potato")

print(vegetables)

print(vegetables.count("cabbage"))

print(vegetables.count("potato"))

Output:

['bottleguard', 'cabbage', 'brinjal', 'bitterguard', 'potato', 'cabbage', 'potato']
2
2

/* Total number of items in the list */

Code:

print(len(vegetables))

Output:

7
