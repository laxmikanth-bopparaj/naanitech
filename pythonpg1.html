Introduction to Python:

What is Python?

Python is a popular programming language. It was created by Guido van Rossum, and released in 1991.

It is used for:

  • web development (server-side)
  • software development
  • mathematics
  • system scripting

Version:

The most recent major version of Python is Python 3

Tools Used:

  • Visual Studio Code
  • Integrated Development Environment, such as Thonny, Pycharm, Netbeans or Eclipse

Features:

Easy to Learn:

Python's syntax is designed for readability and simplicity, this makes it beginner-friendly and promotes maintainable code.

Interpreted Language:

Python code is executed line by line by an interpreter, eliminating the need for a separate compilation step

Platform Independent (Cross-Platform):

Python code can run on various operating systems like Windows, macOS, and Linux without significant modifications

Object-Oriented Programming (OOP):

Python fully supports OOP principles, including classes, objects, inheritance, and polymorphism, enabling the creation of modular and reusable code.
